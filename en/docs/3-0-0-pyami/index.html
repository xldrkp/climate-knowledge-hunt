pyami

An Overview of #semanticClimate Toolkit

The Toolkit consists of several components which work together as a toolchain. There can be more than one toolchain. The pyami tool takes a PDF and converts it into HTML pages. These HTML pages are processed to get structured or semantic HTML. We also add meaning to a lot of the components of the HTML. We can reorganize, search, or annotate the structured HTML.
